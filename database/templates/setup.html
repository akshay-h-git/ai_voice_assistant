Setup Instructions

Follow these simple steps to get your AI Voice Assistant up and running in no time.

1. Download the Assistant

Download the latest version from the link below:

Download Now

2. Extract the Files

Right-click the ZIP file and choose Extract All to unzip the assistant files to your preferred location.

3. Install Requirements

Open a terminal/command prompt in the extracted folder and run:

pip install -r requirements.txt

4. Run the Assistant

In the same terminal, start the assistant with:

python start.py

First make sure that you have registered in the web. Make sure your microphone is working!

5. Customizing AI Voice Assistant and Adding API Key to the AI Voice Assistant

  • Go to Login Page to customize the assistant's wake word, voice settings, and other preferences.
  • Insert your API keys (e.g., for WeatherAPI, Gemini, or other services) after Login under the appropriate fields.
  • Ensure your microphone is enabled and accessible for Python-based applications in your system settings.
